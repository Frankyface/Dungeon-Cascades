# Dungeon Cascades — Failed Approaches

This is the graveyard of dead ends: approaches we actually tried and abandoned. Its
purpose is institutional memory of pain — so a future session does not re-attempt a
path that already cost us time, and knows what to do instead. This file is **never
pruned**; entries stay forever. `handoff.md` links here instead of carrying these
stories itself, keeping the handoff short.

This is distinct from [decisions.md](decisions.md): decisions.md records choices we
made up front (chose X over Y); this file records things we built or attempted and
found to be wrong in practice.

Entry format:

```
## <date> — <what we tried>
**Why it failed:** … · **Do instead:** …
```
